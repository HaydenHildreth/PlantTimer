# PlantTimer

## Dependencies
Python 3
Bottle (package)

Install bottle
```
pip install bottle
```

Clone repository
```
git clone https://github.com/HaydenHildreth/PlantTimer.git
```

Go into folder
```
cd foldername
```

Run python script to launch server
```
python src/main.py
```

Add plants on your new server, at 127.0.0.1:8080
